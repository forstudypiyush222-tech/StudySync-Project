# 📘 StudySync – Learning Project (HTML & CSS)

🔗 **Live Demo:** https://studysync-p.netlify.app  

---

## 🚀 Overview

**StudySync** is a front-end learning project built using only **HTML and CSS**.  
The purpose of this project was to practice layout design, responsive UI, and modern styling techniques by creating a clean and structured study planner interface.

This project focuses purely on **visual design and layout**, without any JavaScript functionality.

---

## 🎯 Purpose

- Practice core HTML structure and semantics  
- Improve CSS layout skills (Flexbox / Grid)  
- Learn responsive design techniques  
- Build a real-world UI without using JavaScript  

---

## ✨ Features

- 🎨 Clean and modern UI design  
- 📱 Fully responsive layout (mobile + desktop)  
- 🧩 Structured sections for study planning  
- 🎯 Focus on visual hierarchy and spacing  
- ⚡ Lightweight and fast  

---

## 🧠 Key Learnings

- Writing semantic HTML  
- Using Flexbox and/or CSS Grid  
- Managing spacing, alignment, and layout  
- Building responsive designs  
- Improving UI/UX fundamentals
